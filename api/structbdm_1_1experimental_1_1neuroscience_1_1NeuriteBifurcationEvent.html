Detailed Description

Contains the parameters to bifurcate a growth cone.

This event is only possible for terminal neurite segments. It creates two new neurite elements and assigns it to daughter left and daughter right of the neurite element that triggered the event (=mother).

Here is the constructor to create a new neurite element for this event NeuriteElementExt::NeuriteElementExt(const NeuriteBifurcationEvent& event, TNeuriteElement* mother, uint64_t new_oid) and the corresponding event handler NeuriteElementExt::EventHandler(const NeuriteBifurcationEvent& event, TDaughter* left, TDaughter* right)
