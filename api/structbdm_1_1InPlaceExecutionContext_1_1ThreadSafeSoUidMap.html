◆ operator[]()

const InPlaceExecutionContext::ThreadSafeSoUidMap::value_type & bdm::InPlaceExecutionContext::ThreadSafeSoUidMap::operator[] ( const SoUid &  uid)

NB: This method fixes any race-condition of ThreadSafeSoUidMap::Insert It might happen that a ThreadSafeSoUidMap::Insert inserts the element into one of the previous_maps_. Therefore, if we don't find the element in map_, we iterate through previous_maps_ to see if we can find it there. This iteration happens only in the unlikely event of a race-condition and is therefore not performance relevant.
