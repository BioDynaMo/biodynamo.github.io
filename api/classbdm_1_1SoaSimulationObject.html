Detailed Description

template<typename TCompileTimeParam, typename TDerived>
class bdm::SoaSimulationObject< TCompileTimeParam, TDerived >

Contains implementation for SimulationObject that are specific to SOA backend. The peculiarity of SOA objects is that it is simulation object and container at the same time.

template<typename TCompileTimeParam , typename TDerived >
std::unordered_map<uint32_t, uint32_t> bdm::SoaSimulationObject< TCompileTimeParam, TDerived >::Commit ( )
inline

This method commits changes made by DelayedPushBack and DelayedRemove. CAUTION:

  • Commit invalidates pointers and references returned by DelayedPushBack.
  • If memory reallocations are required all pointers or references into this container are invalidated
    One removal has constant complexity. If the element which should be removed is not the last element it is swapped with the last one. (CAUTION: this invalidates pointers and references to the last element) In the next step it can be removed in constant time using pop_back.
