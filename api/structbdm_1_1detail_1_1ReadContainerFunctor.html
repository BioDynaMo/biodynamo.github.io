Detailed Description

template<typename TSoSimBackend, typename TBackend>
struct bdm::detail::ReadContainerFunctor< TSoSimBackend, TBackend >

Functor to read SoPointer::so_container_ from a ROOT file. Uses dynamic dispatch to avoid compilation errors between incompatible types. TODO(lukas) Once we use C++17 use if constexpr instead to simplify the logic.
