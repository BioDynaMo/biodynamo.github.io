Detailed Description

template<typename TSimObject>
class bdm::SoPointer< TSimObject >

Simulation object pointer. Required to point to a simulation object with throughout the whole simulation. Raw pointers cannot be used, because a sim object might be copied to a different NUMA domain, or if it resides on a different address space in case of a distributed runtime. Benefit compared to SoHandle is, that the compiler knows the type returned by Get and can therefore inline the code from the callee and perform optimizations.
