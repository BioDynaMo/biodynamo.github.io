Detailed Description

template<typename TCompileTimeParam, typename TDerived>
class bdm::SoaSimObject< TCompileTimeParam, TDerived >

Contains implementation for SimObject that are specific to SOA backend. The peculiarity of SOA objects is that it is simulation object and container at the same time.
