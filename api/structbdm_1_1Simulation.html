Detailed Description

template<typename TCTParam = CompileTimeParam<>>
struct bdm::Simulation< TCTParam >

This is the central BioDynaMo object. It containes pointers to e.g. the ResourceManager, the scheduler, parameters, ...
It is possible to create multiple simulations, but only one can be active at the same time. Creating a new simulation object automatically activates it. Implementation for Simulation can be found in file: simulation_implementation.h. It must be separate to avoid circular dependencies. It can't be defined in a source file, because it is templated.
