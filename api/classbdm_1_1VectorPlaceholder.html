Detailed Description

template<typename T>
class bdm::VectorPlaceholder< T >

VectorPlaceholder has the same interface as a std::vector but no data members. Therefore, there is no dynamic memory allocation and the size of an instance is almost zero. Use case: In a templated class a std::vector data member is not used for all template parameters.
