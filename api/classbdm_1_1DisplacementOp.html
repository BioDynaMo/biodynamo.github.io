bool bdm::DisplacementOp< TSimulation >::force_cpu_implementation_ = false
private

Currently the gpu implementation only supports Spheres. If a simulation contains simulation objects with different shapes with GPU turned on, this wouldn't work. The CPU implementation should be used if this condition is detected. In this case force_cpu_implementation_ will be set to true.
