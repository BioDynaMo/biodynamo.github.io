Detailed Description

template<typename TNeuronSomaSoPtr, typename TNeuriteElementSoPtr>
class bdm::experimental::neuroscience::NeuronNeuriteAdapter< TNeuronSomaSoPtr, TNeuriteElementSoPtr >

The mother of a neurite element can either be a neuron or a neurite. Therefore, this class acts as an intermediate layer that forwards function calls to the correct object.

Template Parameters
TNeuronSomaSoPtrtype of NeuronSoma simulation object pointer
TNeuriteElementSoPtrtype of NeuriteElement simulation object pointer.
