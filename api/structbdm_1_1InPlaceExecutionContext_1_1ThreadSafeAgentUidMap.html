◆ operator[]()

const InPlaceExecutionContext::ThreadSafeAgentUidMap::value_type & bdm::InPlaceExecutionContext::ThreadSafeAgentUidMap::operator[] ( const AgentUid &  uid)

NB: This method fixes any race-condition of ThreadSafeAgentUidMap::Insert It might happen that a ThreadSafeAgentUidMap::Insert inserts the element into one of the previous_maps_. Therefore, if we don't find the element in map_, we iterate through previous_maps_ to see if we can find it there. This iteration happens only in the unlikely event of a race-condition and is therefore not performance relevant.
